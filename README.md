# DSCI-235-Final-Project
DSCI 235 Final Project
## Overview
This project analyzes NFL player salary data to understand how teams allocate salary cap space across positions, experience levels, and teams. The goal is to identify patterns in how value is assigned to players in the league.

## Dataset
- Source: Kaggle (NFL Player Salaries Dataset)
- Includes player attributes such as:
  - Name, Team, Position
  - Age, Years of Experience
  - Height, Weight
  - Cap Hit (salary)

## Key Questions
- Which positions are paid the most?
- How does experience affect salary?
- How do teams differ in their cap allocation strategies?
- Do physical traits relate to salary?

## Methods Used
- Data cleaning with pandas
- Grouping and aggregation (`groupby`)
- Data visualization using matplotlib
- Basic statistical exploration

## Key Findings
- Quarterbacks are by far the highest-paid position.
- Salaries generally increase with experience, peaking mid-career.
- Teams spend similar total amounts due to the salary cap, but differ in allocation strategies.
- Some teams focus heavily on key positions (like QB), while others distribute spending more evenly.
- Physical traits like weight and age have little direct impact on salary.
